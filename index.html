Project Idea:Web application for analysis and decision support on missing persons

This research project is aimed at using the GIS techniques to create a web-based GIS system that will allow querying, storage of spatial data from the spatial database. The system will be able to identify the hotspots for missing children, and apply the spatial autocorrelation and neighbourhood functions and including optimal route analysis in identifying areas that are vulnerable to the cases of missing children, based on the initial cases of abductions in Nairobi.
